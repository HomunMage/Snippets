終於用AI把整個流程串起來：

* 問卷原始檔是 .md
* 自動renfer成前端的問卷系統
* 使用者回答後是json格式
* 送json string 用 AWS Lambda
* AWS Lambda sent to Google API
* doc 末端加上這個json

整個流程在：[https://github.com/posetmage/Survey](https://github.com/posetmage/Survey)
